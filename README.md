# apinf-dashboard-example

> Example dashboard for apinf that uses aggregated data from API-Umbrella.

### Setup

1. Create `config.js` file in project root

  ```
  export default {
    host: '<host>',
    key: '<api_key>',
    token: '<api_token>'
  }
  ```

2. Install dependencies

  ```
  $ npm install && npm start
  ```
